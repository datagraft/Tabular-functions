tabular-functions.datatypes

date-literal
(date-literal n on-error on-null date-format)
Takes a string and converts it to datatype Date in given format. Non-valid values are replaced with integer given as a second 
parameter, null values are replaced with value given as a third parameter.
double-literal
(double-literal n on-error on-null)
Takes a string and converts it to datatype Double. Non-valid values are replaced with value given as a second parameter, 
null values are replaced with value given as a third parameter.
integer-literal
(integer-literal n on-error on-null)
Takes a string and converts it to datatype Integer. Non-valid values are replaced with value given as a second parameter, 
null values are replaced with value given as a third parameter.
